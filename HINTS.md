# Hint Requests

If you're stuck and need a hint, follow this process:

1. Commit your current work: `git commit -m "HINT-REQUEST: [topic]"`
2. Document your thought process below
3. Then open the corresponding file in `/hints/`

Each hint used reduces your score on that section by 5%.
